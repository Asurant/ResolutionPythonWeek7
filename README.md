# Resolution Python Week 7

I made a unit converter that allowed you to convert between the imperial and metric system. I allowed the user to clear the history and rename the window. You are able to directly change the amount with live updates occuring as do so however it will not appear in history and if you wanted to use new units you'd need to click on the convert button. History only gets updated if you use the convert button. Also whyile it make look the same as the original stylesheet for the Qt. It isn't. I just like gray colors for apps.
